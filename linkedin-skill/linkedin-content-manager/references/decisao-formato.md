# Framework de Decisao de Formato

Usado pelo Fluxo: Planejamento Inteligente para decidir o formato ideal.

## Quando usar cada formato

### Texto puro
- Opiniao, reflexao, tendencia
- Storytelling de carreira
- Quando ultimos 2+ posts tiveram imagem ou carrossel

### Texto + Imagem
- Case antes/depois (screenshot, dashboard)
- Post tecnico com diagrama simples
- Quando quer destacar no feed

### Carrossel (PDF)
- Tutorial passo a passo (3+ passos)
- Lista de ferramentas/dicas (5+ itens)
- Maior potencial de salvamento e alcance

### Enquete
- Tema polemico ou com multiplas visoes
- Maximo 1 por semana

## Arvore de Decisao

Tema tem passos ou lista 5+ itens? -> Carrossel
Case com antes/depois visual? -> Texto + Imagem (comparativo)
Tutorial com diagrama/fluxo? -> Texto + Imagem (diagrama)
Ultimos 2+ posts texto puro? -> Texto + Imagem ou Carrossel
Opiniao/reflexao/carreira? -> Texto puro
Default -> Texto + Imagem (destaque)

## Regra de Alternancia

Evitar mais de 2 posts seguidos no mesmo formato.
Distribuicao ideal por mes (8 posts):
- 3 Carrosseis, 3 Texto puro, 1 Texto+Imagem, 1 Enquete
